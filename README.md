## 📰 About UPAD Times

UPAD Times is a monthly digital magazine published by the **UP Association of Detroit** (Est. 1995), celebrating the rich heritage, vibrant culture, and inspiring stories of Uttar Pradesh and its global diaspora.

Each month, we explore a different city from UP, bringing you:
- 📜 **History** — Ancient roots and timeless traditions
- 🏗️ **Development** — Modern transformation and progress
- 🎭 **Culture** — Living traditions and celebrations
- 🍛 **Food** — Culinary heritage and iconic flavors
- 👥 **Community** — Stories from our diaspora

## 🚀 How to Deploy

This repository is configured for **GitHub Pages** automatic deployment.

### Initial Setup
1. Go to **Settings** → **Pages**
2. Set **Source** to: `Deploy from a branch`
3. Set **Branch** to: `main` / `(root)`
4. Click **Save**
5. Site will be live at: `https://upaddetroit.github.io/times/`

### Adding New Issues

1. Create a new folder: `issues/YYYY-MM-cityname/`
2. Add your HTML files to the folder
3. Update `index.html` to add the new issue card
4. Commit and push to GitHub
5. GitHub Pages auto-deploys in ~1 minute

## 📁 File Organization

### Issue Folder Structure
Each issue should follow this pattern:
```
issues/YYYY-MM-cityname/
├── index.html           # Main landing page for the issue
├── development.html     # (optional) Development content
├── history.html         # (optional) History content
├── culture.html         # (optional) Culture content
├── food.html           # (optional) Food content
└── archive.html        # (optional) Archive/navigation
```

### Naming Convention
- **Folders:** `YYYY-MM-cityname` (e.g., `2026-05-varanasi`)
- **Files:** Lowercase, descriptive (e.g., `development.html`, `food.html`)
